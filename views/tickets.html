Ticket Instructions

Step 1

The official Decred GUI wallets Paymetheus (Windows) or Decrediton (macOS/Linux) are the recommended way to use stake pools. All you need to get started is to copy and paste the API Token from the Settings Page. More comprehensive guides are available for Paymetheus and Decrediton. There is also a more general overview of the staking process available here.

It is recommended to use the latest versions of the Decred software before starting.  ⇣ Decred Downloads Page The dcrwallet instructions are the most helpful if you wish to continue on the command-line path.

Step 2

Your multisignature script for delegating votes has been generated. Please first import it locally into your wallet using dcrctl for safe keeping, so you can recover your funds and vote in the unlikely event of a pool failure:

dcrctl --wallet importscript "script"

For example:

$ dcrctl {{ if eq .Network "testnet"}}--testnet{{end}} --wallet importscript {{ .User.MultiSigScript }}

After successfully importing the script into your wallet, you may purchase tickets with voting rights delegated to the pool in either of two ways:

Step 3

Option A - dcrwallet - Automatic purchasing

Stop dcrwallet if it is currently running and add the following to dcrwallet.conf:

[Application Options]
enableticketbuyer=true
pooladdress={{ .User.UserFeeAddr }}
poolfees={{ .PoolFees }}
ticketaddress={{ .User.MultiSigAddress }}
[Ticket Buyer Options]
ticketbuyer.maxpriceabsolute=100

Unlock dcrwallet and it will automatically purchase stake tickets delegated to the pool address.

Option B - dcrwallet - Manual purchasing

Start a wallet with funds available and manually purchase tickets with the following command using dcrctl:

dcrctl {{ if eq .Network "testnet"}}--testnet{{end}} --wallet purchaseticket "fromaccount" spendlimit minconf ticketaddress numtickets poolfeeaddress poolfeeamt

dcrctl {{ if eq .Network "testnet"}}--testnet{{end}} --wallet purchaseticket "default" 100 1 {{ .User.MultiSigAddress }} 1 {{ .User.UserFeeAddr }} {{ .PoolFees}}

Will purchase a ticket delegated to the the multisig address which allows either your or the pool to vote when the ticket is called. This uses funds from the default account only if the current network price for a ticket is less than 100.0 coins.

Step 4 (Optional)

Voting

If you wish to cast votes yourself, please review the guides How To Vote and Solo-Voting. The preferences you set on this stake pool's voting page only affect the stake pool's voting wallets and not your own.
